# Day 1 - Variables and Data Types

## What I Learned
- A variable is used to store data
- Python has different data types like int, float, string, and boolean
- input() is used to get user input
- int() and float() are used to convert input into numbers

## Example Code
name = input("Enter your name: ")
age = int(input("Enter your age: "))

## Reflection
At first, I was confused about data types and input conversion,
but after trying the code, I understood how values are stored and displayed.
