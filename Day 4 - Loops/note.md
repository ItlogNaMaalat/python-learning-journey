# Day 4 - Loops

## What I Learned
- Loops allow repetition of code
- for loops use range() for fixed iterations
- while loops depend on conditions
- break stops the loop
- continue skips an iteration

## Exercises
- Printed even numbers using loops
- Built a password retry system
- Calculated sum of numbers
- Created a number guessing game
- Generated star patterns

## Reflection
Loops are powerful and reduce repetitive code.
